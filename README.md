## Setup

### 1. Install Dependencies

```bash
npm install
```

### 2. Configure Contentstack

Create a `.env` file in the root directory with your Contentstack credentials:

```env
REACT_APP_CONTENTSTACK_API_KEY=your_api_key_here
REACT_APP_CONTENTSTACK_DELIVERY_TOKEN=your_delivery_token_here
REACT_APP_CONTENTSTACK_ENVIRONMENT=your_environment_here
REACT_APP_CONTENTSTACK_REGION=us
```

### 3. Contentstack Content Models

Create the following content types in your Contentstack account:

#### Movie
- Title (Text)
- Slug (Autogenerated)
- Description (Rich Text)
- Release Year (Number)
- Duration (Text)
- Genre (Reference - multiple)
- Director (Reference - single)
- Poster Image (File)
- Banner Image (File)
- Rating (Number)
- Trailer URL (URL)
- Featured (Boolean)

#### Director
- Name (Text)
- Slug (Autogenerated)
- Biography (Rich Text)
- Profile Image (File)
- Birth Year (Number)
- Movies Directed (Reference - multiple)

#### Genre
- Name (Text)
- Slug (Autogenerated)
- Description (Rich Text)
- Featured Image (File)

#### Review
- Movie (Reference)
- Reviewer Name (Text)
- Rating (Number)
- Review Text (Multi-line Text)
- Date (Date/Time)

#### Chatbot Prompts (Optional)
- Movie (Reference)
- Prompt (Text)
- Response (Multi-line Text)

### 4. Run the Application

```bash
npm start
```

The application will open at [http://localhost:3000](http://localhost:3000)
